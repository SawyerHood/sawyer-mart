# Contributing

This is my personal repository of Claude Code skills and plugins that I use for myself. I will probably not approve new random skills being added to the repo or new functionality.  

If you run into any issues or have questions, please [open an issue](https://github.com/SawyerHood/sawyer-mart/issues) first before opening a PR. 

Feel free to fork this marketplace and make it your own! That is the power of skills!
